# 企業登録フォーム 完全実装コード

このドキュメントには、KAMUKAMUを参考にした企業登録フォームの完全なコードが含まれています。

## ファイル: order/signup.php

提供されたプロンプトの完全なコードをここに記載します。
サーバーにアップロードする際は、このコードをorder/signup.phpとして保存してください。

### 主要機能:
1. 企業・お届け先情報入力
2. ご登録者情報入力
3. 郵便番号から住所自動入力
4. パスワード強度チェック
5. 利用規約表示
6. フォームバリデーション

### 実装済み機能:
- Material Design + Bootstrap 5.1.3
- レスポンシブデザイン
- 郵便番号API連携（zipcloud）
- パスワード表示/非表示切替
- リアルタイムバリデーション

---

## ファイル: order/api/signup_api.php

企業登録処理APIの完全なコードです。

### 主要機能:
1. POSTデータ受信・バリデーション
2. メールアドレス重複チェック
3. トランザクション処理
4. 企業コード自動生成（3桁英字）
5. ユーザーコード生成（企業コード + 0001）
6. パスワードハッシュ化（bcrypt）
7. 自動ログイン（セッション設定）

### データベーステーブル:
- companies: 企業情報
- users: ユーザー情報（最初のユーザーはcompany_admin）
