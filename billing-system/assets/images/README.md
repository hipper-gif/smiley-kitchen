# 画像ファイルディレクトリ

## 店舗ロゴ

領収書に表示する店舗ロゴを配置してください。

### ファイル名
- `logo.png`

### 推奨サイズ
- 最大幅: 150px
- 最大高さ: 80px
- 透過PNG推奨

### アップロード方法
1. ファイルサーバーにFTPまたはファイルマネージャーでアクセス
2. `billing-system/assets/images/` ディレクトリに移動
3. `logo.png` という名前でロゴファイルをアップロード

### 注意事項
- ロゴファイルはGitHubには保存されません（.gitignoreで除外）
- サーバーへの直接アップロードが必要です
- ロゴがない場合でも領収書は正常に表示されます
